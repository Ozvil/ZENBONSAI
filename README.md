
# ZenBonsai App v0.4.1 (PWA + Vercel)

## Desarrollo
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy (Vercel)
- Con CLI:
```bash
npm i -g vercel
vercel
vercel --prod
```
- O desde el dashboard importando este repo. Vercel detectará Vite; `vercel.json` ya está configurado.
